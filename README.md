# 함께Walk 미니게임

> **산책길 모험** 미니게임  
> React + TypeScript + Vite + HTML5 Canvas 기반 모바일 캐주얼 러닝게임

---

## 개요

산책길을 달리며 발자국을 수집하고 장애물을 피하는 힐링 미니게임입니다.

---

## 핵심 게임 루프

```
터치 드래그 or ← → 키로 캐릭터 좌우 이동
→ 발자국 수집 (+10점 / 파워워커·달빛길 중 +20점)
→ 물병 10개 수집 → 파워워커 발동 (5초 무적 + 속도 1.7배 + 점수 2배)
→ 시계 수집 → 6초간 속도 절반
→ 장애물 충돌 → 게임오버
→ 화면 고정 후 탭 → 결과 확인
```

---

## 아이템

| 아이템    | 효과                                         |
| --------- | -------------------------------------------- |
| 🟡 발자국 | +10점 (파워워커·달빛길 중 +20점)             |
| 💧 물병   | 게이지 +1, 10개 모으면 파워워커 발동         |
| ⏱️ 시계   | 6초간 전체 속도 0.5배 (단풍길 350m부터 등장) |

## 주요 기능

### 게임 시스템

- **캐릭터 선택** — 여자 3종 / 남자 3종, 원형 아바타, 마지막 선택 기억
- **조작** — 터치 드래그 + 키보드 ← → 방향키 지원
- **파워워커 모드** — 테마별 오브 궤도 이펙트 + 비네트 + 화면 경고 (달빛길은 황금 별 오브)
- **속도 시스템** — 최대 5배 캡 (눈길 도달 시점), 달빛길은 2배 고정으로 여유롭게
- **속도 슬로우** — ⏱️ 수집 시 6초간 모든 속도 절반 (달빛길에선 등장 안 함)
- **말풍선 시스템** — 게임 시작 / 파워워커 / 장애물 조우 / 달빛길 전용 메시지

### 산책 여행 시스템 (6테마 + 보너스 1)

| 거리   | 테마               | 장애물                          |
| ------ | ------------------ | ------------------------------- |
| 0m~    | 🌳 공원 산책길     | 바위, 물웅덩이                  |
| 150m~  | 🌲 숲길            | 그루터기, 다람쥐 dodger, 웅덩이 |
| 350m~  | 🍁 단풍길          | 바위, 단풍낙엽                  |
| 600m~  | 🌸 벚꽃길          | 바위, 벚꽃낙엽                  |
| 1000m~ | ❄️ 눈길            | 바위, 눈사람                    |
| 1500m~ | 🎋 대나무숲길      | 죽순🎍, 돌맹이, 등산객 dodger   |
| 2000m~ | 🌙 달빛길 ✨보너스 | 장애물 없음, 황금 발자국만 등장 |

### 살아있는 장애물 (dodger)

- **다람쥐** (숲길) — 190px 이내 접근 시 "!" 뜨며 반대 방향으로 통통 튀며 도망
- **등산객** (대나무숲) — "먼저 가세요~!" 말풍선 후 비켜줌
- 도망 여부와 관계없이 **충돌 시 게임오버**

### 달빛길 진입 인트로 컷신 🌙

2000m 도달 시 자동 재생되는 컷신 (`CompletionIntro` 클래스):

```
① 황금 발자국 5개 등장 → "어? 황금 발자국?!" 말풍선
② 캐릭터가 발자국을 먹으며 위로 이동
③ 화면 위로 퇴장 → 페이드 아웃
④ 검은 화면: 캐릭터가 아래서 올라옴 → 발자국 → 달빛문 등장
⑤ "발자국을 따라가보자!" → 유저가 직접 발자국 수집 후 문 진입
⑥ 화이트 플래시 → 달빛길 진입 + 완주 토스트 표시
```

- 달빛문 내부: 방사형 빛줄기 + ✦ 반짝이 파티클 + 황금 외곽 발광
- 발자국 미수집 상태로 문 접근 시 힌트 말풍선 표시
- 말풍선 탭으로 즉시 진행 / 자동 진행 혼용

### 달빛길 보너스 구간 🌙

- 2000m 도달 시 진입 — 장애물 없음, 황금 발자국만 등장 (+20점)
- 전용 BGM(`bgm_finished.mp3`)으로 자동 전환
- 별·유성·사이드 반짝이 + 파워모드 시 황금 별 오브 이펙트
- **90초 보너스 타이머** 후 FINISHED 배너 + 포토타임 진입
- 포토타임: 캐릭터 중앙 정렬, 아이템 숨김, 완주 말풍선 고정 표시 (파워모드 이펙트 숨김)
- 완주 오버레이에서 랭킹 등록 → 달빛길로 돌아가기 → 메인 이동

### 산책 도감 🗺️

- 테마 구간 최초 진입 시 자동 해금
- 도감 카드 클릭 → 실제 테마 배경 애니메이션 + 장애물 미리보기 팝업
- 달빛길은 금색 점선 테두리로 **보너스 구간** 표시
- 달빛길 해금 시 달 이모지 float 애니메이션 + 골드 글로우

### 산책 모드 🚶

- **전체 테마(6개) 해금 시 활성화** — 도감에서 각 테마 탭 후 산책하기
- 속도 고정 (🌳1.0x ~ 🎋5.0x, 하단 ＋/－ 버튼으로 조절)
- 상하좌우 자유 이동 (드래그 / 키보드 ↑↓←→)
- 아이템 스폰 없음 — 테마 감상에 집중
- 테마별 색상 간판 + 산책 분위기 말풍선
- 달빛길 포토타임: 자유 이동 + 말풍선 직접 편집 가능

### 게임 방법 안내 팝업

- 메인 화면 '게임 방법 보기' 버튼으로 진입
- 기본 조작 / 아이템 / 파워워커 / 특별한 이웃들 / 장애물(실제 이미지) / 테마 구간 안내
- 히든 테마 도전 배너 (동기부여)

### 기록 & 랭킹

- **전체 랭킹** — Firebase Realtime Database, 상위 50위
- **점수순 / 거리순 탭** 분리 — 각각 독립 갱신 (점수 신기록이면 점수만, 거리 신기록이면 거리만 업데이트)
- 닉네임 기기 ID 보호 — 같은 닉네임은 동일 기기에서만 업데이트 가능
- 욕설 필터링, 마지막 닉네임 자동 기억

### 사운드

- **메인 BGM** — 게임 시작 시 자동 재생, 게임오버 시 중지 (loop)
- **달빛길 BGM** — 2000m 진입 시 자동 전환, 완주 결과 확인 시 종료
- **물병 SFX** — Web Audio API 재생 (iOS 버벅임 방지, 첫 터치 시 AudioContext 언락)

### 성능 최적화

- **emitUpdate 스로틀링** — React 리렌더 60fps → ~10fps, 아이템 수집은 즉각 반영
- **in-place 배열 관리** — filter() 제거로 GC 압박 해소 (매 프레임 6개 배열 생성 없음)
- **iOS 오디오 최적화** — HTMLAudioElement.play() → AudioBufferSourceNode로 메인 스레드 블로킹 제거

### 로딩 스플래시

- JS 실행 전부터 즉시 표시 — 초기 검정 깜빡임 없음
- 앱 로고 + 발자국 걷기 애니메이션 (좌·우·좌 순서로 하나씩)
- 최소 1.2초 노출 후 0.35초 페이드 아웃

## 환경 변수 설정

프로젝트 루트에 `.env.local` 파일 생성:

```
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_DATABASE_URL=...
VITE_FIREBASE_PROJECT_ID=...
```

Firebase Realtime Database 규칙:

```json
{
  "rules": {
    "rankings": {
      ".read": true,
      ".write": true,
      ".indexOn": ["score", "nickname", "distanceMeters"]
    }
  }
}
```


## 설계 원칙

- 게임 로직과 UI 상태 분리 — `GameEngine` 클래스 내부는 순수 게임 로직
- 상태관리 라이브러리 없음 — `useState` (UI) + `useRef` (게임 실시간 데이터)
- 모바일 우선 — 세로 화면, 터치 드래그, `passive: false` 이벤트
