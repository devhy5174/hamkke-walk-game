# 함께Walk

모바일 웹에서 즐길 수 있는 캐주얼 러닝 게임입니다.

캐릭터를 좌우로 움직이며 발자국을 모으고, 장애물을 피해서 높은 점수에 도전하는 게임입니다.

## 프로젝트 목적

이 프로젝트는 React + TypeScript + Vite 기반으로 간단한 모바일 게임을 만들어보는 실습용 프로젝트입니다.

추후 토스 인앱 또는 모바일 웹 환경에서 실행할 수 있는 미니게임 형태를 목표로 합니다.

## 주요 기능

- 모바일 세로 화면 최적화
- 캐릭터 좌우 이동
- 발자국 아이템 수집
- 장애물 피하기
- 점수 시스템
- 게임오버 및 다시하기
- Canvas 기반 게임 화면

## 기술 스택

- React
- TypeScript
- Vite
- HTML5 Canvas

## 실행 방법

```bash
npm install
npm run dev
```
